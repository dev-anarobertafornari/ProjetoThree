# Projeto de Computação Gráfica — Universo Inspirado nos Vingadores

## 📌 Sobre o Projeto

Este projeto de **Computação Gráfica** foi desenvolvido utilizando **Node.js** e **Three.js**, com o objetivo de recriar um ambiente 3D inspirado no complexo dos Vingadores.

A proposta vai além de uma simples reprodução: o cenário apresenta personagens do universo Marvel em um contexto alternativo, fora da narrativa original, proporcionando uma experiência visual criativa e educativa.

---

## 🌇 Cenário e Ambientação

O ambiente foi construído para transmitir uma atmosfera cinematográfica:

* 🌅 Céu com **degradê de cores simulando o pôr do sol**
* ☀️ Esfera iluminada posicionada próxima ao horizonte, representando o sol
* 🏙️ Estrutura inspirada no complexo dos Vingadores
* 🎭 Personagens com **animações**, criando mais imersão no cenário

---

## 🎮 Funcionalidades

* Renderização 3D em tempo real com Three.js
* Iluminação e ambientação dinâmica
* Inserção de personagens estilizados
* Animações básicas em alguns elementos
* Experiência visual interativa
